# A.N Ride

A.N Ride es una plataforma privada de transporte entre ciudades de Texas.

## Ciudad base

Austin, Texas

## Vehículo inicial

SUV

## Teléfono del negocio

512-749-4195

## Rutas iniciales

- Austin → Houston: $200

- Austin → Dallas: $200

- Austin → San Antonio: $100

## Partes del sistema

1. App para pasajeros

2. App para conductores

3. Panel de administración

4. Página web oficial

5. Sistema de reservas

6. GPS y mapas

7. Pagos

8. Base de datos de clientes

## Administrador

El propietario tendrá acceso con usuario y contraseña para:

- Ver viajes

- Cambiar tarifas

- Administrar clientes

- Administrar conductores

- Ver ingresos

- Aceptar o rechazar solicitudes
