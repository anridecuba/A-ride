# A.N Ride - Plan del Proyecto

## Objetivo

Crear una plataforma de transporte entre ciudades de Texas.

## Fase 1

- Panel de administración

- Inicio de sesión

- Registro de clientes

- Registro de conductores

- Tarifas

- Reservas

## Fase 2

- GPS en tiempo real

- Notificaciones

- Historial de viajes

- Reportes

## Fase 3

- Aplicación para iPhone

- Aplicación para Android

- Pagos con tarjeta

- Publicación
