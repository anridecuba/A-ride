# A.N Ride - Tipos de Usuarios

## 1. Administrador

El administrador es el dueño de A.N Ride.

Puede:

- Ver todos los viajes

- Aceptar o rechazar solicitudes

- Cambiar tarifas

- Ver clientes

- Ver conductores

- Ver ingresos

- Agregar vehículos

- Cambiar configuración

## 2. Pasajero

El pasajero es el cliente que solicita un viaje.

Puede:

- Crear cuenta

- Iniciar sesión

- Solicitar viaje

- Ver precio

- Pagar

- Ver historial

- Calificar el servicio

## 3. Conductor

El conductor realiza los viajes.

Puede:

- Iniciar sesión

- Ver solicitudes disponibles

- Aceptar o rechazar viajes

- Ver la ruta

- Finalizar el viaje

- Ver ganancias
